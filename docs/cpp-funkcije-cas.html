Мотивација (5 мин)

Поставити питање: „Шта ако се део кода понавља? Можемо ли то избећи?“ Увести појам функције као "мини-програма" који решава један задатак и може се позивати више пута.

Понављање: шта је main()?

Функција main() је почетна тачка извршавања програма — рачунар увек прво покреће код унутар main().

Кључна значења
  • int main() — враћа целобројну вредност (обично 0 као успех)
  • return 0; — означава успешан завршетак програма
  • Тело функције између { } садржи наредбе које се извршавају

Аналогија: main() је као главна соба у кући — из ње улазимо у друге просторије (функције).

#include <iostream>
using namespace std;

int main() {
    cout << "Zdravo svete!" << endl;
    return 0;
}

Зашто користимо функције

  1. Подела програма на мање логичке целине
  2. Поновна употреба кода
  3. Јасноћа и прегледност
  4. Лакше одржавање

Пример: уместо да често пишемо if (broj % 2 == 0), дефинишемо:

bool jeParan(int broj) {
    return broj % 2 == 0;
}

и користимо је на више места:

if (jeParan(a)) ...
if (jeParan(b)) ...
if (jeParan(c)) ...

Излагање новог градива (пример)

Синтакса функције:

tip_vracene_vrednosti ime_funkcije(parametri) {
    // telo funkcije
}

Пример целог програма:

#include <iostream>
using namespace std;

bool jeParan(int broj) {
    if (broj % 2 == 0)
        return true;
    else
        return false;
}

int main() {
    for (int i = 1; i <= 10; i++) {
        if (jeParan(i))
            cout << i << " je paran broj." << endl;
        else
            cout << i << " je neparan broj." << endl;
    }
    return 0;
}

Увежбавање (15–20 мин)

Задатак 1

Напиши функцију bool deljivSaTri(int broj) која враћа true ако је број дељив са 3. У main() испиши бројеве од 1 до 20 и означи који су дељиви са 3.

Задатак 2 (напредније)

Напиши функцију int zbirParnih(int n) која враћа збир свих парних бројева од 1 до n. У main() учитати n и приказати резултат.

Домаћи задатак

Напиши функцију која прима два броја и враћа већи. У main() тражи од корисника да унесе пет парова бројева и за сваки испиши који је већи.

Закључак (5 мин)

  • main() је почетак програма
  • Функције деле програм на мање целине
  • Функције могу имати параметре и вратити вредност
  • for и if се могу користити унутар функције
