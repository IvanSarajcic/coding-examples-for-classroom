C++ Библиотеке - Преглед

Библиотека Опис Пример
#include <iostream> Омогућава улаз и излаз преко конзоле. cin (унос са тастатуре), cout (испис на екран)
#include <iomanip> Служи за форматирање излаза (ширина поља, број децимала, поравнање…). setprecision(2) за приказ бројева са две децимале
#include <cmath> Садржи математичке функције: квадратни корен, степен, синус, косинус… sqrt(x), pow(a, b), sin(x)
#include <string> Омогућава рад са стринговима (текстуалним подацима). string ime = "Petar";
#include <ctime> Користи се за рад са временом и датумом. Често заједно са rand() да би случајни бројеви били „различити". srand(time(0));
#include <vector> Омогућава рад са векторима (динамички низови). vector<int> brojevi = {1, 2, 3};
#include <algorithm> Садржи корисне алгоритме за рад са подацима. sort(brojevi.begin(), brojevi.end());
#include <fstream> Омогућава рад са фајловима (читање и писање у датотеке). ofstream file("test.txt"); file << "Zdravo";
#include <numbers> Садржи математичке константе као што су π (pi), e, златни пресек итд. (C++20 и новије) std::numbers::pi
