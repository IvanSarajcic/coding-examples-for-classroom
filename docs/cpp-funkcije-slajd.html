ЗАДАТАК: ФУНКЦИЈА

1. Задатак

  • Отвори Dev-C++ / VS Code.
  • Напиши функцију bool jeParan(int x) која враћа true ако је број паран.
  • У main() направи петљу од 1 до 10.
  • За сваки број испиши нпр: 7 је НЕПАРАН или 12 је ПАРАН.
  • Користи позив jeParan(i) унутар петље.
  • Тестирај програм (мењај границе ако желиш).

2. Шаблон

#include <iostream>
using namespace std;

bool jeParan(int x) {
    return x % 2 == 0; // врати true ако је x паран
}

int main() {
    for (int i = 1; i <= 10; i++) {
        if (jeParan(i))
            cout << i << " je PARAN\n";
        else
            cout << i << " je NEPARAN\n";
    }
    return 0;
}
Изазов (+): Прошири опсег (нпр. 1–50) и преброј колико је парних бројева.

Провери: функција ради? Петља позива функцију? Формат исписа јасан?
